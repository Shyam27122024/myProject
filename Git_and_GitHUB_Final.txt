

1) Create GITHUB project (During create project - Click on ReadME file button)

2) Create a local Git (Project Space )- folder  Automation Project 
3) in Automation Project ( Initiate Git ) - Git init 
4) Git clone URL 
5) After cloning URL - moves in project by using- >  cd (Actul Project name )  
6) make changes in files or add new files 
7) check git status 
8) Git add . / git add * / git add filename.extention 
9) Git commit -m " Message" 
10) Git push origin main                        ( main branch existing during creation Repo ) same in local
